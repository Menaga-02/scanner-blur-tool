# Scanner Blur Tool

This tool allows you to upload a scanned image and blur a specified scanner area.

## How to Run

1. Install dependencies:
```
pip install flask opencv-python
```

2. Run the application:
```
python app.py
```

3. Open in your browser:
```
http://127.0.0.1:5000
```

4. Upload your image → Blur → Download.
